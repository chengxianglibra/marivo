Marivo 将人类定义的数据语义，与 agent 执行的分析流程分开。

- [语义层](/zh-cn/v0.3/concepts/semantic-layer/) 解释 datasources、domains、entities、dimensions、metrics 和 semantic refs。
- [分析流程](/zh-cn/v0.3/concepts/analysis-workflow/) 解释 sessions、operators 和 typed frames。
- [就绪检查](/zh-cn/v0.3/concepts/readiness/) 解释为什么未完成的 semantic objects 会在分析前被阻止。
- [证据链](/zh-cn/v0.3/concepts/evidence/) 解释 operator outputs 如何通过 findings、propositions 和 assessments 被追踪。
