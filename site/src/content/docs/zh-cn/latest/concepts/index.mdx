---
title: 核心概念
description: Marivo semantic modeling 和 analysis 的核心词汇。
---

Marivo 将人类定义的数据语义和 agent 执行的分析流程分开。

- [Semantic layer](/zh-cn/latest/concepts/semantic-layer/) 解释 datasources、domains、entities、dimensions、metrics 和 semantic refs。
- [Analysis workflow](/zh-cn/latest/concepts/analysis-workflow/) 解释 sessions、operators 和 typed frames。
- [Readiness](/zh-cn/latest/concepts/readiness/) 解释为什么未完成的 semantic objects 会在分析前被阻止。
- [Evidence](/zh-cn/latest/concepts/evidence/) 解释 operator outputs 如何通过 findings、propositions 和 assessments 被追踪。
