Marivo separates the human-authored meaning of data from the analysis workflow an agent executes.

- [Semantic layer](/en/v0.3/concepts/semantic-layer/) explains datasources, domains, entities, dimensions, metrics, and semantic refs.
- [Analysis workflow](/en/v0.3/concepts/analysis-workflow/) explains sessions, operators, and typed frames.
- [Readiness](/en/v0.3/concepts/readiness/) explains why incomplete semantic objects are blocked before analysis.
- [Evidence](/en/v0.3/concepts/evidence/) explains how operator outputs are traced through findings, propositions, and assessments.
