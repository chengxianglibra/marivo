Marivo is a Python library, not a hosted service. It gives AI agents a typed semantic layer, a metric-centered analysis workflow, and an auditable evidence trail for analytical conclusions.

Marivo is not a text-to-SQL wrapper. Python declarations are the contract, ibis expressions are the execution language, and typed frames are the analysis boundary between steps.

<CardGrid>
  <Card title="Unified semantics" icon="seti:data">
    Datasources, entities, metrics, dimensions, and relationships are declared in Python and addressed through semantic refs.
  </Card>
  <Card title="Metric-centered analysis" icon="seti:python">
    Agents start from trusted metrics and compose observe, compare, decompose, correlate, forecast, and quality checks.
  </Card>
  <Card title="Trustworthy evidence" icon="seti:config">
    Operators record findings, propositions, and assessments so results can be traced and reviewed.
  </Card>
  <Card title="Readiness gates" icon="seti:lock">

  </Card>
