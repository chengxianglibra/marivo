The primary user journey lives under **Get started** and **Work with an agent**:
begin at [Get started](/en/latest/), then continue with
[First agent-guided analysis](/en/latest/first-analysis/) and the
[business-question guide](/en/latest/guides/business-question/). Use this section
as an integration and reference map when you need to inspect the contracts behind
that journey.

- [Semantic Layer](/en/latest/concepts/semantic-layer/) — semantic object and
  authoring contracts.
- [Analysis Workflow](/en/latest/concepts/analysis-workflow/) — analysis
  operators, typed frames, artifacts, and recovery behavior.
- [Readiness](/en/latest/concepts/readiness/) — report fields and explicit
  certification for semantic changes.
- [Evidence](/en/latest/concepts/evidence/) — typed findings, bounded artifact
  digests, inference boundaries, and audit traces.
- [Python API Reference](/api/) — exact public modules, classes, functions, and
  signatures for implementation work.
