# Phase 6.1 — Runtime Self-Sufficiency Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Demolish `app/service.py` (`SemanticLayerService`, 3193 lines) and `app/session/session_manager.py` (`SessionManager`, 20KB). After this plan ships, `MarivoRuntime` is fully self-sufficient — `__init__(ports, core)` only, no `wire_svc()`, no `_svc` field. Local profile keeps working end-to-end.

**Architecture:** Move logic in this order — (A) extend port surfaces with three list/projection methods; (B) build the missing local `SqliteStepStore` and `FileArtifactStore` adapters that the demolition will depend on; (C) create three `runtime/` use case modules that absorb the service/session-manager logic; (D) switch `MarivoRuntime` methods to delegate via these modules; (E) repoint `AppServices` and the seven API endpoints; (F) delete the now-orphaned classes and add an import-linter tripwire. `MarivoRuntime` keeps its flat-method facade; use case modules take `runtime: MarivoRuntime` as their first argument.

**Tech Stack:** Python ≥3.11, SQLite (WAL mode), FastAPI, SQLAlchemy Core (server adapters only), pytest, pydantic, `import-linter`. Repository entrypoints: `make test`, `make typecheck`, `make lint`, `make format`. Never invoke bare `python` / `pytest` / `mypy` / `ruff`.

**Spec:** [`docs/superpowers/specs/2026-05-08-phase6.1-runtime-self-sufficiency-design.md`](../specs/2026-05-08-phase6.1-runtime-self-sufficiency-design.md)

## Pre-implementation Audit Findings

The writing-plans audit (`grep app/service.py`, `grep app/session/session_manager.py`, code reads) surfaced scope that is not visible in the spec but is necessary for demolition:

1. **`StepStore` Protocol does not have its own port file** — it lives inside `app/ports/artifact_store.py`. Plan splits it into `app/ports/step_store.py` to match the parent spec §6 layout.
2. **No local `SqliteStepStore` / `FileArtifactStore` exists.** Local mode currently routes step / artifact writes through `_svc._insert_step()` and `_svc._commit_artifact_with_extraction()` (private methods), bypassing the port. `RuntimePorts.artifact_store` and `RuntimePorts.step_store` are still `Optional[None]`. Demolition is impossible without these adapters.
3. **`SemanticLayerService` has 13 methods not listed in the spec migration table.** Each requires a destination decision (Stage F has a dedicated audit task).
4. **`MarivoRuntime` has only 14 `_svc.*` call sites** (10 intent dispatchers + 4 session methods + 1 svc property). The 15 semantic helpers in spec §5.2 are mostly called from `app/intents/*.py` directly via `service.compile_step(...)` rather than via Runtime methods. Migration of those helpers happens by repointing intent module imports, not by changing Runtime methods.
5. **Seven API endpoints touch `services.service.*`:** `app/api/health.py:18` (`discover_catalog`) and six in `app/api/sessions.py` (lines 83, 114, 228, 253, 281, 306).
6. **`AppServices` shape:** `app/api/app_factory.py:115` constructs it. Holds both `service: SemanticLayerService` and `runtime: MarivoRuntime` plus seven other fields. After demolition, drop `service`.
7. **Ten test files import `SemanticLayerService` or `SessionManager` directly.** `tests/test_session_manager.py` is the class-level test file (delete with the class). The rest use them as fixtures and need fixture rewrites to use `MarivoRuntime`.

```python
from __future__ import annotations

import json
import sqlite3
from datetime import UTC, datetime
from pathlib import Path
from typing import Any

from app.contracts.ids import SessionId, StepId
from app.contracts.session import Step


class SqliteStepStore:
    """SQLite-backed StepStore. Shares the local state.db with SessionStore."""

    def __init__(self, db_path: Path) -> None:
        self._db_path = db_path
        self._db_path.parent.mkdir(parents=True, exist_ok=True)
        self._ensure_schema()

    def insert_step(
        self,
        step_id: StepId,
        session_id: SessionId,
        step_type: str,
        summary: str,
        result: dict[str, Any],
        *,
        provenance: dict[str, Any] | None = None,
        semantic_metadata: dict[str, Any] | None = None,
    ) -> None:
        conn = self._connect()
        try:
            conn.execute(
                "INSERT INTO steps (step_id, session_id, step_type, summary, "
                "result, provenance, semantic_metadata, created_at) "
                "VALUES (?, ?, ?, ?, ?, ?, ?, ?)",
                (
                    str(step_id),
                    str(session_id),
                    step_type,
                    summary,
                    json.dumps(result, sort_keys=True),
                    json.dumps(provenance, sort_keys=True) if provenance else None,
                    json.dumps(semantic_metadata, sort_keys=True) if semantic_metadata else None,
                    datetime.now(UTC).isoformat(),
                ),
            )
            conn.commit()
        finally:
            conn.close()

    def list_steps(self, session_id: SessionId) -> list[Step]:
        conn = self._connect()
        try:
            rows = conn.execute(
                "SELECT step_id, session_id, step_type, summary, result, "
                "provenance, semantic_metadata, created_at "
                "FROM steps WHERE session_id = ? ORDER BY created_at ASC, step_id ASC",
                (str(session_id),),
            ).fetchall()
            return [
                Step(
                    step_id=StepId(row[0]),
                    session_id=SessionId(row[1]),
                    step_type=row[2],
                    summary=row[3],
                    result=json.loads(row[4]),
                    provenance=json.loads(row[5]) if row[5] else None,
                    semantic_metadata=json.loads(row[6]) if row[6] else None,
                    created_at=row[7],
                )
                for row in rows
            ]
        finally:
            conn.close()

    def _connect(self) -> sqlite3.Connection:
        conn = sqlite3.connect(str(self._db_path))
        conn.execute("PRAGMA journal_mode=WAL")
        conn.execute("PRAGMA busy_timeout=5000")
        conn.execute("PRAGMA foreign_keys=ON")
        return conn

    def _ensure_schema(self) -> None:
        conn = self._connect()
        try:
            conn.execute(
                """CREATE TABLE IF NOT EXISTS steps (
                    step_id TEXT PRIMARY KEY,
                    session_id TEXT NOT NULL,
                    step_type TEXT NOT NULL,
                    summary TEXT NOT NULL,
                    result TEXT NOT NULL,
                    provenance TEXT,
                    semantic_metadata TEXT,
                    created_at TEXT NOT NULL
                )"""
            )
            conn.execute(
                "CREATE INDEX IF NOT EXISTS idx_steps_session "
                "ON steps (session_id, created_at)"
            )
            conn.commit()
        finally:
            conn.close()
```

Create `app/adapters/local/file_artifact_store.py`:

```python
from __future__ import annotations

import json
from datetime import UTC, datetime
from pathlib import Path
from typing import Any
from uuid import uuid4

from app.contracts.ids import ArtifactId, SessionId, StepId
from app.evidence_engine.finding_extractors import FindingExtractorRegistry


class FileArtifactStore:
    """File-backed ArtifactStore.

    Layout:
      <root>/<session_id>/<step_id>.json     (committed artifact)
      <root>/<session_id>/_index.jsonl       (append-only index for list_artifacts)

    The on-disk layout is internal to the adapter; consumers go through the
    ArtifactStore Protocol.
    """

    def __init__(self, root: Path) -> None:
        self._root = root
        self._root.mkdir(parents=True, exist_ok=True)

    def insert_artifact(
        self,
        session_id: SessionId,
        step_id: StepId,
        artifact_type: str,
        name: str,
        content: Any,
        *,
        lifecycle: str = "committed",
        artifact_schema_version: str | None = None,
    ) -> ArtifactId:
        aid = ArtifactId(f"art-{uuid4()}")
        record = {
            "artifact_id": str(aid),
            "session_id": str(session_id),
            "step_id": str(step_id),
            "artifact_type": artifact_type,
            "name": name,
            "lifecycle": lifecycle,
            "artifact_schema_version": artifact_schema_version,
            "content": content,
            "created_at": datetime.now(UTC).isoformat(),
        }
        self._write_artifact(record)
        self._append_index(record)
        return aid

    def commit_artifact_with_extraction(
        self,
        session_id: SessionId,
        step_id: StepId,
        artifact_type: str,
        name: str,
        content: Any,
        *,
        step_type: str | None = None,
        artifact_schema_version: str | None = None,
    ) -> ArtifactId:
        # Mirror the server behavior: run extraction and store the side-effect
        # alongside the primary artifact. The registry call is identical.
        registry = FindingExtractorRegistry()
        extracted = registry.extract(
            artifact_type=artifact_type,
            content=content,
            step_type=step_type,
        )
        aid = self.insert_artifact(
            session_id,
            step_id,
            artifact_type,
            name,
            content,
            artifact_schema_version=artifact_schema_version,
        )
        if extracted:
            self.insert_artifact(
                session_id,
                step_id,
                f"{artifact_type}.extracted",
                f"{name}.extracted",
                extracted,
            )
        return aid

    def resolve_artifact_for_ref(
        self,
        session_id: SessionId,
        step_id: StepId,
    ) -> dict[str, Any] | None:
        path = self._artifact_path(session_id, step_id)
        if not path.exists():
            return None
        return json.loads(path.read_text())

    def resolve_artifact_id_for_step(
        self,
        session_id: SessionId,
        step_id: StepId,
    ) -> ArtifactId | None:
        record = self.resolve_artifact_for_ref(session_id, step_id)
        if record is None:
            return None
        return ArtifactId(record["artifact_id"])

    def resolve_artifact_with_id(
        self,
        session_id: SessionId,
        step_id: StepId,
    ) -> tuple[ArtifactId, dict[str, Any]] | None:
        record = self.resolve_artifact_for_ref(session_id, step_id)
        if record is None:
            return None
        return ArtifactId(record["artifact_id"]), record

    def list_artifacts(self, session_id: SessionId) -> list[dict[str, Any]]:
        index = self._index_path(session_id)
        if not index.exists():
            return []
        return [json.loads(line) for line in index.read_text().splitlines() if line.strip()]

    # --- internals ---

    def _session_dir(self, session_id: SessionId) -> Path:
        d = self._root / str(session_id)
        d.mkdir(parents=True, exist_ok=True)
        return d

    def _artifact_path(self, session_id: SessionId, step_id: StepId) -> Path:
        return self._session_dir(session_id) / f"{step_id}.json"

    def _index_path(self, session_id: SessionId) -> Path:
        return self._session_dir(session_id) / "_index.jsonl"

    def _write_artifact(self, record: dict[str, Any]) -> None:
        path = self._artifact_path(SessionId(record["session_id"]), StepId(record["step_id"]))
        tmp = path.with_suffix(f".tmp-{uuid4()}")
        tmp.write_text(json.dumps(record, sort_keys=True))
        tmp.replace(path)

    def _append_index(self, record: dict[str, Any]) -> None:
        index = self._index_path(SessionId(record["session_id"]))
```
