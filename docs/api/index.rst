Marivo Python API Reference
===========================

Marivo's public Python surface is three modules. Each page groups the symbols
that module's ``__all__`` exports into thematic sections, generated from the
package's Google-style docstrings. Every symbol links to its own page.

.. list-table::
   :widths: 25 75
   :header-rows: 0

   * - :doc:`datasource <datasource>`
     - Connect to, inspect, and register data sources.
   * - :doc:`semantic <semantic>`
     - Declare entities, dimensions, measures, and metrics.
   * - :doc:`analysis <analysis>`
     - Run metric-centered analysis over the semantic layer.
